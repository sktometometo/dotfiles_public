# dotfiles_public

dotfiles関連のファイルと設定スクリプト類

## Ubuntu 環境構築手順

- Ubuntu をインストール
- パッケージ類をアップデート
- ドライバをインストール

```bash
sudo apt update
sudo ubuntu-drivers install
```

- 以下のソフトウェアをインストール,環境構築
  + dropbox
  + keepassxc
  + slack
  + Google chrome
  + inkscape
  + Vivaldi browser

- docker + nvidia-dockerのインストール
- CUDAのインストール
